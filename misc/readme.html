Bowser Network Readme

To change HOSTNAME:

hostname echo > /etc/hostname

STATIC IP TIME:

cp /etc/netctl/examples/ethernet-static /etc/network.d/

Edit the file (vim!) with the appropriate IP, Gateway, and DNS

Disable any other netcfg profiles

If you're not sure if there are:

netctl list

Back to disabling...

systemctl stop netcfg@ systemctl disable netcfg@

Enable the new profile

systemctl enable netcfg@ systemctl start netcfg@

The network should be correctly set up after reboot
